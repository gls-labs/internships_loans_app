# Models - README

## Document

- Represents the template documents that are linked to a Loan Type. It's an uploadable and
  downloadable file with a title that will be copied as Loan Documents when a Loan is
  initialized.

## Loan Type

- Categorization for the loans a Lender provides. It defines which documents are associated
  to a Loan when initialized.

## Loan

- This is the work instance of a Loan Process. It's the gathering point for the Lender, the
  borrower and the documents to be worked on.

## Loan Document

- They are the different documents that the Borrower must provide, and where that person
  will upload them. It will also be the place where comments interactions happen.
